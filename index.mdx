## Why Milvus?

Milvus separates compute from storage and scales each tier independently. It supports all major vector index algorithms — HNSW, IVF, DiskANN, SCANN — along with GPU acceleration via NVIDIA CAGRA.

## Get started in three steps

<Steps>
  <Step title="Install pymilvus">
    Install the Python SDK to connect to Milvus Lite (embedded), Standalone, or Zilliz Cloud.
    ```bash
    pip install -U pymilvus
    ```
  </Step>
  <Step title="Create a collection and insert vectors">
    Define a schema with a vector field and ingest your embeddings.
    ```python
    from pymilvus import MilvusClient

    client = MilvusClient("milvus_demo.db")  # Milvus Lite local file

    client.create_collection(
        collection_name="my_collection",
        dimension=768,
    )

    client.insert(
        collection_name="my_collection",
        data=[{"id": 1, "vector": [0.1] * 768, "text": "Hello Milvus"}],
    )
    ```
  </Step>
  <Step title="Search for similar vectors">
    Run an approximate nearest neighbor (ANN) search and retrieve results with metadata.
    ```python
    results = client.search(
        collection_name="my_collection",
        data=[[0.1] * 768],  # query vector
        limit=5,
        output_fields=["text"],
    )
    print(results)
    ```
  </Step>
</Steps>
