## Embedding models

Milvus is embedding-model agnostic. You can generate vectors outside Milvus and pass them in, or use `pymilvus[model]` to call embedding APIs directly in your ingestion pipeline.

  </Card>
  <Card title="HuggingFace" icon="face-smile">

  <Card title="Cohere" icon="bolt">

  <Card title="Sentence Transformers" icon="arrow-right-arrow-left">

  </Card>
</CardGroup>

### pymilvus[model]

`pymilvus[model]` is a utility layer bundled with the Milvus Python SDK. It wraps embedding model APIs and local model runtimes behind a consistent interface so you can swap providers without rewriting your ingestion code.

```bash
pip install "pymilvus[model]"
```

```python
from pymilvus.model.dense import OpenAIEmbeddingFunction

ef = OpenAIEmbeddingFunction(model_name="text-embedding-3-small", api_key="YOUR_KEY")
vectors = ef.encode_documents(["Milvus stores vectors", "Fast approximate nearest neighbor search"])
```

<Tip>
  `pymilvus[model]` supports dense embeddings, sparse embeddings (BM25, SPLADE), and reranking models. See the [embeddings documentation](https://milvus.io/docs/embeddings.md) for the full list of supported providers.
</Tip>
